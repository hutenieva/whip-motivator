# Whip Motivator

A fun Chrome extension that adds a physics-based whip cursor to Google Meet calls. Crack it at your screen to point things out during screen sharing.

## Features

- Realistic rope physics — flexible pink whip with rigid brown handle
- Click anywhere to strike that exact spot with a crack animation and sound
- 4 hit modes: Neutral (sparks), Loving, Angry, Mixed (emojis)
- Toggle on/off with `Alt+W` or the toolbar button
- Works only on meet.google.com

### Manual install
1. Clone this repo
2. Open `chrome://extensions` in Chrome
3. Enable **Developer mode** (top right)
4. Click **Load unpacked** and select the repo folder

## Usage

1. Join a Google Meet call
2. Click the Whip Motivator icon in the toolbar and toggle it on
3. Move your mouse — the whip trails behind your cursor
4. Click anywhere to crack the whip at that spot
5. Switch hit modes in the popup for different effects
6. Press `Alt+W` to quickly toggle off when needed
